# VanteoPay_BillPay
Repository to manage Bill Pay Module of VanteoPay

Users can:
- Add a Payee with a name and address
- Create a Payment
- Make a one time payment
- View estimated delivery of the payment
- View list of payees 
- Schedule a future payment to a payee
- Setup automatic recurring payments
- Select a Biller from a list of billers
- Link an Invoice to a Biller with an Account Number
- View upcoming payments due to a biller
- dismiss a payment due
- make a payment to a Biller
- View list of Billers
- View history of payments
- View details of a Biller
- View details of a Payee
- 
---

## Product Vision
Create a trusted mobile banking app for foriegn nationals arriving in the United States. Bill Pay allows users to pay servic providers directly from the mobile banking app to manage all their bills from one central location.
